#  e340.差分練習

## 內容
給定一陣列B，請輸出這個陣列的差分陣列A。
定義1 : 當B[i]為A[0]+A[1]+...A[i]，B陣列為A陣列的前綴和。
定義2 : 當B陣列為A陣列的前綴和，A陣列為B陣列的差分陣列。
 


---

### 輸入說明
輸入的第一行有一個整數N (1 <= N <= 200000)，代表B陣列大小。
第二行有N個整數以空白分隔，依序表示B[0], B[1], B[2] ... B[N-1]。
陣列中數字的絕對值不會超過10^9。

### 輸出說明
輸出一行，其中有N個整數以空白分隔，依序表示A[0], A[1], A[2] ... A[N-1]。
