#  e521.106 彰雲嘉區複賽 - Q1 三角形

## 內容
一、三角形 ( 10分 )
   
    給定三個正整數代表三條線段的長度，請寫一程式判斷此三條線段是否能圍成一個三角形，若可以圍成一個三角形，請進一步判斷是否能切成兩個相同的直角三角形。
 
* 測資均為官方測資
* 為模擬正式競賽，WA 時 不公開正確答案!
* 加油~ !


---

### 輸入說明
輸入資料中第一列為一整數n，代表接下來有n組測試資料。
第二列開始每列有三個正整數 A,B和C(0<A, B, C <= 10000)，代表三條線段的長度。

### 輸出說明
如果A , B 和C無法圍成一個三角形，請輸出 0，
如果A , B 和C可以圍成一個三角形，請輸出 1，再進一步判斷此三角形是否能切成兩個相同的直角三角形，若可則再輸出一個1，若否則再輸出一個0。
