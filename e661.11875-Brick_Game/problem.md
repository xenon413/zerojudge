#  e661.11875 - Brick Game

## 內容
在孟加拉的一個村莊中，堆磚塊遊戲非常流行。堆磚塊遊戲是團隊遊戲，每個團隊由奇數個玩家組成。玩家數量必須大於1，但不能大於10。每個玩家的年齡必須在11和20之間。任何兩個玩家都不能具有相同的年齡。每個團隊都有一個隊長。兩個玩家之間的溝通差距取決於他們的年齡差異，如果年齡差異較大，溝通差距就會更大。因此，他們以這樣一種方式選擇隊長，以使隊中比隊長年輕的球員人數等於比隊長年長的球員人數。現在提供團隊所有成員的年齡。您必須確定隊長的年齡。


---

### 輸入說明
輸入第一行為一個整數T (T ≤ 100)，代表測資數量。每組測資一行，每行有一個整數N (1 < N < 11)，代表團隊成員的數量，接著有N個整數，代表團隊所有成員的年齡。這N個整數皆介於11到20(包含)之間。年齡將以嚴格遞增的順序或嚴格遞減的順序給出。

### 輸出說明
對於每組測資，輸出"Case x：a"，x為測資編號，a為隊長的年齡。
