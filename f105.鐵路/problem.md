#  f105.鐵路

## 內容
Tony老師在某一次經過一個特殊的火車站時，看到這個火車站鐵軌的奇特設計，總共有三條鐵軌$A、B$,
現在火車從 A 方向來，預定從 B 方向離開。火車共有 N 節車廂（N <=1000），並且各車廂依次以1到 N 來編號。 你可以讓火車依序往 A 方向的鐵軌開過去，但是一旦一台火車進站後，就不能再回到 A 方向的鐵軌上了，並且一旦離開車站往 B 方向後，也不能再回到車站。
相對位置如下圖所示，現在在 $A$ 鐵軌上有 $n$ 台火車，由前到後分別編號$1、2、...、n$ ,Tony老師現在想要給你一組數字，為$1, 2, 3, 4,..., n$的任意排列。他想知道火車可不可以按照該排列到 $B$ 鐵軌上
 



---

### 輸入說明
輸入一個整數$n$，代表有$n$台火車，接著輸入$n$個數字$a_1、a_2、...、a_n$,代表Tony老師想知道的火車排列
$n <= 1000$

### 輸出說明
若可以達到Tony老師的要求，請輸出Yes，否則輸出No
