#  h081.1. 程式交易

## 內容
小明最近想要用程式做股票交易，給一個股票的歷史價格 $a[1], a[2], ..., a[n]$ 他的投資策略如下1. 同一個時間最多只會持有一張股票, 並會在時間點 $1$ 花 $a[1]$ 買進。2. 若當下持有股票且該股票買進價格為 $x$，當遇到價格 $y$ 大於等於 $x + D$ 時即賣出，並轉得利潤 $y - x$。3. 若當下沒有持有股票且上一次的賣出價格為 $x$，當遇到價格 $y$ 小於等於 $x - D$ 時則會買進股票。
輸出依照上述規則買賣後所得到的利潤和，若交易結束仍持有股票，則不考慮該股票買進的成本，直接無視該股票即可。


---

### 輸入說明
第一行輸入兩個正整數 $n, D$，接下來有 $n$ 個正整數，代表每個時間點股票的價格。
 
數字範圍
$1 \leq n \leq 100, 1 \leq D \leq 100$$1 \leq a[i] \leq 100$
子題配分
(50%): $n = 3$(50%): 無額外限制

### 輸出說明
輸出一個正整數，代表總利潤。
