#  c638.天干地支

## 內容
天干，是中國古代的一種文字計序符號，共10個字: 甲、乙、丙、丁、戊、己、庚、辛、壬、癸，循環使用。( wiki )
地支是指木星軌道被分成的十二個部分，記為子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥。木星的公轉周期大約為十二年，所以中國古代用木星來紀年，故而稱為「歲星」。後來又將這十二個部分命名，這就是「地支」。( wiki )
