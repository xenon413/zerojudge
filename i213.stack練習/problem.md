#  i213.stack 練習

## 內容
請你實作stack相關的幾個基本操作：
在堆疊頂端加入元素。輸出堆疊最頂端的元素。刪除堆疊最頂端的元素。


---

### 輸入說明
第一行有一個 N(1≤N≤105) ，接下來有 N 行，每一行一開始有一個 k ，代表哪一種操作：
如果 k=1 ，請再讀入一個整數 x(1≤x≤109) ，並在堆疊頂端加入該整數。如果 k=2 ，請輸出堆疊最頂端的元素，如果當前堆疊內沒有元素，請輸出 −1 。如果 k=3 ，請刪除堆疊最頂端的元素，如果當前堆疊內沒有元素，請無視該操作。

### 輸出說明
對於每次 k=2 操作，輸出一個整數於一行，代表當時堆疊最頂端的元素。如果當時堆疊內沒有元素，請輸出 −1 。
