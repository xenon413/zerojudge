#  d448.好熱好熱

## 內容
國中理化的熱學裡常有一種問題:有一位天才非常自以為的寫下了自己的溫標。X ，給你一個t。C ，叫你換成多少。X 比如：100。C = 150。X    0。C = -50。X  50。C = ？。X正解：50//=======================這個問題一直困擾了許多人很久大家都知道，程式是來解決生活中的問題的所以，這個工作我們就交給電腦來跑吧!//==================


---

### 輸入說明
每筆測資有五個數，依序為t1 、t2 、t3 、x1 和 x3(我們保證t1 > t2 > t3 且 x1 > x3)t1。C 相當於 x1。Xt3。C 相當於 x3。X請你算出t2。C 相當於多少。X，也就是 x2

### 輸出說明
x2
